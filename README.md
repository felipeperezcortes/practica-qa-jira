# Práctica QA en Jira – Testing Manual y Funcional

Este repositorio documenta mi práctica aplicada utilizando Jira para testing manual y funcional, simulando flujos de trabajo reales de QA utilizados en equipos profesionales.

## Alcance
- Creación y documentación de reportes de bugs
- Testing funcional de flujos críticos (login, registro y transferencias)
- Gestión del ciclo de vida de bugs en Jira (Open → In Progress → Fixed → Retesting → Closed)
- Retesting y comentarios QA profesionales
- Clasificación de incidencias (Bug vs Improvement)
- Aplicación de criterios de severity y priority
